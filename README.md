# *Assets* da disciplina MATA65

Aqui voce poderá encontrar todos arquivos necessários para rodar os códigos exemplo dos Labs. Temos:

1. arquivos css para as página dos Labs
2. O código base do *framework Three.JS* - essa será a versão que utilizaremos até o final do semestre 
3. Modelos 3D
4. Texturas
5. Imagens

Esse repositório poderá ser atualizado ao longo do semestre para acrescentar novos Assets para as atividades de Lab e/ou trabalhos. 

## Importante:   

Para que os códigos dos Labs encontrem esse repositório voce deve ajustar a sua localização de acordo com o servidor *web* que voce vai utilizar: 

- [*VSCode Live Server*](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) ou 
- [*Python Web Server*](https://realpython.com/python-http-server/).

Consulte a documentação específica do seu servidor *web* e em caso de dúvidas consulte seu professor buscando auxílio.
